ElasticDino

Dense Semantic Features using Elastic Deformations

Ulysse Mizrahi Sean Zaretzky Amit Bermano Dov Danon
Tel Aviv University
Paper Code

Abstract

Vision transformers (ViTs) are the backbone of countless vision tasks, including dense ones, such as segmentation and depth estimation. However, the quadratic complexity of self-attention limits semantic feature resolution and induces spatial artifacts. Dense prediction hence requires significant task-specific, computational and data collection efforts. In this work, we introduce ElasticDino, an adapter for these backbones, which produces high-resolution feature maps that preserve the fine details of the input image. As a key insight, we observe that all channels display consistent artifacts, which can be simultaneously resolved and matched to the input image structure. Using a lightweight, fully convolutional architecture predicting only spatial deformations, ElasticDino produces pixel-aligned, artifact-free, full-resolution feature maps. We choose deformations for their ability to efficiently affect channels consistently and strictly preserve sharp semantics, avoiding otherwise inherent smoothness biases. Given the lack of ground truth, we employ pixel-level linear heads to predict a set of downstream dense tasks, guiding the deformation process. ElasticDino enables efficient and high-performance training of dense tasks — we demonstrate superior performance to current methods, and further obtain state-of-the-art results on semantic segmentation over a range of popular benchmarks. We argue that ElasticDino is recommended for many of the dense tasks currently employing pretrained attention-based backbones.

Motivation

DINO features are low resolution and prone to artifacts

Standard ViT backbones operate on image patches, often yielding 32x32 or 64x64 feature grids for images that downstream models need to reason about at full resolution. This introduces positional noise, feature bleeding across boundaries, and extra burden on task-specific decoders.

Method

Task-guided Elastic Deformations

ElasticDino alternates bilinear upsampling with lightweight Feature Deformation layers. Each layer reads the current semantic feature map and the image at the same resolution, predicts a 2D deformation field with depthwise-separable convolutional blocks, and warps the features instead of directly rewriting their values.

FD

Shared Deformations

Each Feature Deformation layer predicts a 2D warp field from the image and current feature map. The same spatial warp is applied to all channels, matching the channel-consistent structure of ViT artifacts while keeping computation lightweight.

SP

Semantic Preservation

The network warps existing features rather than directly rewriting feature values. Sampling from the original feature map preserves backbone semantics, discourages hallucinated task labels, and sharpens boundaries without smoothing away object identity.

PH

Pixel-Level Heads

ElasticDino trains without ground-truth high-resolution DINO features. Instead, simple 1x1 task heads supervise every pixel for RGB, depth, semantic segmentation, normals, shading, albedo, residuals, and intrinsic decomposition.

Because the heads operate independently at each pixel, they cannot hide spatial mistakes with decoder context. The deformation layers must therefore align the frozen semantic features to image structure before the heads can solve the dense tasks.
